# Paper

NeurIPS-format preprint draft (`main.tex`) for the THINGS odd-one-out alignment
study. Numbers are filled from the real 5-model run (Table 1: mean±std over seeds
{0,1,2}); see `../results/REPORT.md` and `../EXPERIMENT_LOG.md` (review board).

## Build

The document compiles **as-is** (clean `article` fallback, placeholder figures).
For the real NeurIPS look and figures:

1. **Get the NeurIPS style** (one file): download `neurips_2024.sty` from
   <https://neurips.cc/Conferences/2024/PaperInformation/StyleFiles> and place it
   in this folder. Without it, the paper still builds in a plain article style.
2. **Copy the figures** generated by the pipeline:
   ```bash
   mkdir -p figures
   cp ../results/transfer_heatmap.png ../results/rsa.png figures/
   ```
   (If absent, the paper renders labeled placeholder boxes.)
3. **Compile:**
   ```bash
   make            # latexmk -> main.pdf
   # or:
   pdflatex main && bibtex main && pdflatex main && pdflatex main
   ```
